# Portfólio - Adérito De Barros
Versão 2.0 (Template Around) - Dezembro 2025

Este repositório contém o código fonte do portfólio pessoal de Adérito De Barros.

## Estrutura
- `index.html`: Arquivo principal (Portfólio atual).
- `index-old.html`: Backup do portfólio anterior (Bootstrap vanilla).
- `around.createx.studio/`: Diretório contendo os assets do tema (CSS, JS, Fonts).
- `assets/`: Imagens e logos personalizados.

## Manutenção
### Adicionar Projetos
Procure pela seção `<div class="masonry-grid" data-columns="2">` no `index.html` e adicione um novo bloco `<article>` seguindo o padrão existente.

### Alterar Tecnologias
Na seção de Habilidades (Flip Cards), edite as listas `<ul>` dentro do verso dos cards (`card-flip-back`).

### Cores
O tema usa variáveis CSS baseadas no Bootstrap. Para ajustes rápidos, edite `assets/css/style.css` (se criado) ou use classes utilitárias do Bootstrap (ex: `text-primary`, `bg-dark`).
